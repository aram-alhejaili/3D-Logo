# Pokemon Logo 3D

A 3D model of the Pokemon logo created using Onshape.

## Design

The model consists of:

- A black outer frame
- A red upper section
- A white lower section
- A central black ring and white button

## Tools Used

- Onshape
- 3D Sketching
- Extrude
- Appearance and color editing

## File

The 3D model is available in STL format and uses millimeters as the measurement unit.
